# Requirements

## Functional Requirements

- Simulate real-world customer support tasks
- Provide 3 difficulty levels (easy, medium, hard)
- Implement OpenEnv interface (reset, step, state)
- Provide reward between 0 and 1
- Include programmatic grading system
- Provide baseline inference script

## Non-Functional Requirements

- Deployable on HuggingFace Spaces
- Docker support
- Low latency response
- Reproducible results

## Data Requirements

- Predefined support queries dataset
- Keyword-based expected responses
